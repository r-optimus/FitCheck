
# FitCheck

## Beschreibung

FitCheck ist eine Web-Anwendung zur Analyse persönlicher Fitness- und Gesundheitsdaten. Nutzer können verschiedene Kennzahlen wie BMI, Kalorienbedarf, Wasserbedarf und Proteinbedarf berechnen sowie ihre Fortschritte langfristig verfolgen.

Zusätzlich bietet die Anwendung Workout-Empfehlungen, tägliche Fitness-Challenges sowie eine Benutzerverwaltung mit Login und Registrierung.

---

## Funktionen

* BMI-Berechnung und BMI-Bewertung
* Berechnung des täglichen Kalorienbedarfs
* Berechnung des Wasser- und Proteinbedarfs
* Individuelle Workout-Empfehlungen
* Daily Challenges
* Benutzerregistrierung und Login
* Fortschrittsverfolgung mit Speicherung von Gesundheitsdaten
* Export von Ergebnissen
* Hell- und Dunkelmodus

## Installation und Start

### 1. Projekt bereitstellen

Das Projekt liegt als ZIP-Archiv vor und ist vor der Ausführung in ein lokales Verzeichnis zu extrahieren.

### 2. Abhängigkeiten installieren

```bash
pip install -r requirements.txt
```

### 3. Anwendung starten

```bash
python app.py
```

### 4. Anwendung aufrufen

Nach dem Start ist die Anwendung unter folgender Adresse erreichbar:

```text
http://localhost:8080
```
